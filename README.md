# The Dark Room — Photography Website

A premium photography portfolio website for The Dark Room agency, founded by Muskan.

## Tech Stack
- Pure HTML/CSS/JavaScript (no frameworks)
- Three.js for 3D card carousel
- WebP images for performance
- Scroll-driven animations

## Features
- 3D photo album cards with scroll navigation
- All 4 albums: B&W Street, Organic Silence, Tactile Concrete, Crafted Perfection
- Stats counter, Process section, Studio section
- Lightbox with full-res image viewer
- Progress bar, frosted glass nav, animated loader
- Mobile responsive
- UK GDPR/cookie consent compliant

## Albums
- `images/bw/` — B&W Street photography
- `images/organic-silence/` — Conceptual photography
- `images/tactile-concrete/` — Aerial & Drone photography
- `images/crafted-perfection/` — Product photography
